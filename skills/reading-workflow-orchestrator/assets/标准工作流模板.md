# 标准工作流模板

# 阅读工作流

## 一、任务名称
填写当前任务名称。

## 二、输入材料
列出当前已有材料：
- 书籍 / 文章 / 演讲 / 访谈 / 股东信
- 截图 / 笔记 / Markdown
- 已完成草稿 / 阶段性整理

## 三、最终目标
明确最终要达成什么：
- 看懂内容
- 做读书笔记
- 写公众号文章
- 写小红书笔记
- 做课程 / PPT
- 做知识库
- 做长期研究专题

## 四、当前阶段判断
当前属于哪一步：
- 原始材料阶段
- 初步整理阶段
- 分析提炼阶段
- 改写发布阶段
- 知识沉淀阶段

## 五、推荐 skill 路由

### 第一步
调用：
- book-deconstruction
或
- author-thinking-analysis

目标：
说明第一步要解决什么问题。

产物：
说明第一步产出什么。

### 第二步
调用：
- article-rewriting
或
- knowledge-base-formatting

目标：
说明第二步要解决什么问题。

产物：
说明第二步产出什么。

### 第三步（如有）
调用：
- knowledge-base-formatting

目标：
说明最后沉淀什么。

产物：
说明最终形成什么知识资产。

## 六、阶段性产物建议

### 分析层产物
- 核心问题
- 结构图
- 核心观点
- 关键概念
- 方法
- 案例
- 逻辑链

### 输出层产物
- 标题
- 正文
- 摘要
- 朋友圈文案
- 小红书版本

### 沉淀层产物
- 索引页
- 节点页
- 标签
- 双链
- 文件夹结构

## 八、一句话执行建议
用一句话说明当前最该做的事。
