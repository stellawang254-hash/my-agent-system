# Skill: Author Thinking Analysis

## Description
把一个人物、作者、企业家、投资人、管理者的书、文章、股东信、访谈、演讲等材料，
从零散内容整理成一套“可调用的思想地图”。

重点不是介绍人物生平，
也不是简单摘录金句，
而是提炼：

1. 长期稳定的核心观点
2. 反复强调的主题
3. 看问题的方式
4. 做判断的逻辑
5. 方法偏好与价值排序
6. 思想是否有演变

## When to use
- 帮我拆这个人的思想体系
- 帮我总结他长期稳定的观点
- 帮我提炼他的决策逻辑
- 帮我分析他反复强调的主题
- 帮我整理这个作者的思想地图
- 帮我从多篇材料中提炼核心认知

## Workflow

### Step 1：确认研究对象和边界
- 研究对象是谁
- 当前材料有哪些
- 用户最想研究哪一面
- 材料够不够支撑整体判断

### Step 2：识别高频主题
找出反复出现的主题词和判断。

### Step 3：建立思想主轴
如：
- 如何看世界
- 如何做判断
- 如何做选择
- 如何理解风险
- 如何看待长期回报

### Step 4：提炼核心观点
说明：
- 观点本身是什么
- 为什么这样判断
- 出现在哪些材料里
- 为什么是高频稳定观点
- 容易误解成什么
- 对今天有什么价值

### Step 5：提炼判断逻辑
说明：
- 出发点是什么
- 最在意哪些变量
- 如何处理不确定性
- 如何做取舍
- 如何避免犯错

### Step 6：提炼方法偏好
说明：
- 方法名
- 解决问题
- 核心步骤
- 适用边界
- 常见误区

### Step 7：建立时间线
整理：
- 早期重点
- 中期强化
- 后期变化
- 始终不变的点

### Step 8：提炼代表性案例
说明：
- 案例是什么
- 为什么有代表性
- 验证了什么思想

### Step 9：输出“可调用版本”
整理成：
- 核心主题索引
- 观点卡
- 决策逻辑卡
- 方法卡
- 案例卡
- 时间线
- 可写作素材清单

## Default Output Modes
### Mode A：思想地图版
- 这个人最核心的思想主题
- 他反复强调的 5-8 个关键词
- 他的核心判断方式
- 他的决策逻辑
- 他的代表性案例
- 哪些观点是长期稳定的
- 哪些观点有阶段性变化
- 这套思想今天还有什么价值
- 对普通人最有启发的 3-5 点
- 一句话总结思想底色

## One-line Positioning
不是总结一个人说过什么，
而是提炼这个人长期稳定的思想操作系统。
