# 🧠 My Agent System

一个围绕「读书拆解 → 思想分析 → 内容输出 → 知识沉淀」构建的本地 AI 能力系统。

一句话：

👉 不是写内容，而是把内容变成可以反复调用的能力。

---

## 🚀 我在做什么

这不是一个简单的提示词集合，而是一套：

- 可复用的能力模块（Skills）
- 可拆解复杂任务的工作流（Workflow）
- 可持续积累的知识体系（Knowledge Base）
- 可直接产出内容的生产系统

目标很明确：

👉 **让 AI 从“聊天工具”，变成“能干活的系统”**

---
